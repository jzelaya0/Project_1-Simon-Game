SIMON

How to Play

An addictive and fun memory game. An enhancement of the famous “Simon Says” Test your memory and boost your brain power with this great game. Watch the lights and try to repeat the sequence. The sequence will get longer with each round. This great retro game can be enjoyed by all ages.

Challenge Mode

Ready for a challenge? Challenge mode let's you play regular Simon, but with a twist. For each round the board will rotate randomly. It might turn left or it might turn right. It is all up to the board! You can enable challenge mode before you start the game or during your current game. Have fun and good luck!
